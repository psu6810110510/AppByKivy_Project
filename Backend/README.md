# Backend - Sudoku Engine

โฟลเดอร์นี้เปรียบเสมือนสมองของเกม Kivy Sudoku รับผิดชอบในส่วนของตรรกะเกม การสร้างโจทย์ และการตรวจสอบความถูกต้อง

# โครงสร้างข้อมูล (Data Structure)
กระดาน Sudoku ของเราถูกเก็บข้อมูลในรูปแบบ 2D Array (List of Lists) ขนาด 9x9
* ตัวเลข `1-9` หมายถึง ช่องที่มีตัวเลขอยู่
* ตัวเลข `0` หมายถึง ช่องว่างที่รอการเติม

# กฎและหลักการทำงานพื้นฐาน


ในการตรวจสอบว่าผู้เล่นสามารถใส่ตัวเลขลงในช่องว่างได้หรือไม่ (หรือตอนที่ Engine กำลังพยายามสร้างโจทย์) ระบบจะตรวจสอบผ่านกฎเหล็ก 3 ข้อของ Sudoku ซึ่งทำงานอยู่ในฟังก์ชัน `is_safe(board, row, col, num)`:

1. Row Validation (`_is_valid_row`): ตัวเลขที่จะเติม ต้องไม่ซ้ำกับตัวเลขอื่นใน แถว เดียวกัน
2. Column Validation (`_is_valid_col`): ตัวเลขที่จะเติม ต้องไม่ซ้ำกับตัวเลขอื่นใน คอลัมน์ เดียวกัน
3. Box Validation (`_is_valid_box`): ตัวเลขที่จะเติม ต้องไม่ซ้ำกับตัวเลขอื่นใน กล่องย่อยขนาด 3x3
