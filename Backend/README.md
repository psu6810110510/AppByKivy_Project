# Backend - Sudoku Engine

นี่คือ สมอง ของโปรเจกต์ Kivy Sudoku ที่จัดการ Logic ของเกมทั้งหมด ตั้งแต่การสร้างโจทย์, การแก้สมการ, ไปจนถึงระบบช่วยเหลือผู้เล่น

## การจัดการสถานะ (State Management)
ในคลาส `SudokuEngine` จะมีการเก็บข้อมูลกระดานแยกกัน 3 ส่วน เพื่อไม่ให้ข้อมูลตีกัน:
* `self.board`: **กระดานปัจจุบัน** ที่ผู้เล่นเห็น มีช่องว่าง และอัปเดตตลอดเวลาที่ผู้เล่นพิมพ์
* `self.solution`: **กระดานเฉลย** ตารางที่สมบูรณ์ 100% ซ่อนไว้หลังบ้านเพื่อใช้ตรวจคำตอบ
* `self.initial_puzzle`: **โจทย์เริ่มต้น** เก็บค่าตอนที่เพิ่งทำช่วงว่างเสร็จใหม่ๆ เอาไว้ใช้เวลาผู้เล่นกดปุ่ม Reset

## อธิบายฟังก์ชันหลัก (Core Functions)

### 1. ระบบสร้างโจทย์ (The Generator)
* **`generate_board(difficulty)`**
  * **หน้าที่:** สร้างโจทย์ Sudoku ใหม่ตามระดับความยาก ("Easy", "Medium", "Hard")
  * **การทำงาน:** เติมเลขแนวทแยง -> ใช้ Solver แก้จนเต็ม -> เซฟเก็บเป็น `solution` -> เจาะรู (ลบเลขออก) ตามความยาก -> เซฟเก็บเป็น `initial_puzzle` -> ส่ง `board` ให้ผู้เล่น
* **`remove_numbers(k)`**
  * **หน้าที่:** สุ่มลบตัวเลขออกจากกระดานจำนวน `k` ช่อง ยิ่งลบเยอะ เกมยิ่งยาก

### 2. ระบบคำนวณและตรวจสอบ (The Solver & Validator)
* **`solve_sudoku(board)`**
  * **หน้าที่:** AI หาคำตอบให้กระดานที่ว่างอยู่ โดยใช้ **Backtracking Algorithm** (สุ่มเดินหน้า ถ้าตันให้ถอยหลัง)
* **`is_safe(board, row, col, num)`**
  * **หน้าที่:** ตรวจสอบกฎเหล็ก 3 ข้อ (ห้ามซ้ำในแถว, ห้ามซ้ำในคอลัมน์, ห้ามซ้ำในกล่อง 3x3)

### 3. ระบบสำหรับผู้เล่น (Gameplay Mechanics)
ฟังก์ชันเหล่านี้เตรียมไว้ให้ฝั่ง Frontend เรียกใช้เมื่อผู้เล่นกดปุ่มต่างๆ:
* **`check_move(row, col, num)`**
  * **หน้าที่:** ตรวจว่าเลขที่ผู้เล่นเพิ่งพิมพ์ลงไป ตรงกับ `solution` หรือไม่ (คืนค่า True/False)
* **`get_hint()`**
  * **หน้าที่:** หาช่องที่ยังเป็นช่องว่าง (เลข 0) สุ่มเลือกมา 1 ช่อง แล้วเอาเลขที่ถูกต้องจากเฉลยมาเติมให้
  * **คืนค่า:** พิกัดและตัวเลข `(row, col, correct_num)` เพื่อให้ Frontend เอาไปแสดงผล
* **`reset_puzzle()`**
  * **หน้าที่:** ล้างตัวเลขที่ผู้เล่นกรอกไปทั้งหมด แล้วคืนค่ากระดานกลับไปเป็น `initial_puzzle` ตอนเริ่มเกม
* **`is_game_won()`**
  * **หน้าที่:** ตรวจสอบว่าผู้เล่นเติมตัวเลขครบทุกช่องและถูกต้องตามเฉลยทั้งหมดหรือไม่ ใช้สำหรับแสดงป็อปอัป "คุณชนะแล้ว!"
