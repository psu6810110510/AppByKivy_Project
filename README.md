ภาพรวมของโปรแกรม การทำงานใน Code และ การรันตัวเกม Project : Kivy Sudoku 
1. ภาพรวมของโปรแกรม 
Kivy Sudoku  คือแอปพลิเคชันเกมลับสมองซูโดกุที่พัฒนาด้วยภาษา Python โดยใช้เฟรมเวิร์ก Kivy ในการสร้างหน้าจอแสดงผล โปรแกรมนี้ออกแบบมาตามหลักการแยกส่วนการทำงาน  ระหว่างส่วนตรรกะการคำนวณ(Backend) และส่วนแสดงผล(Frontend) เพื่อความง่ายในการพัฒนาและเพิ่มฟีเจอร์ในอนาคต

2. ฟีเจอร์ที่โดดเด่น 
2.1 Difficulty Algorithm หรือ ระบบสุ่มโจทย์ตามระดับความยาก (Easy, Medium, Hard) โดยใช้ Algorithm ในการลบตัวเลขออกจากกระดานที่สมบูรณ์แล้ว 
2.2 Pencil Mode (Note)ระบบจดบันทึกตัวเลขขนาดเล็กในช่องว่าง 
2.3 Toggle สลับเปิด/ปิดเลขทดสอบ ช่วยให้ผู้เล่นวิเคราะห์ความเป็นไปได้หลายตัวเลข
2.4 ระบบย้อนกลับและทำซ้ำการเดินผ่าน Stack เพื่อความผิดพลาดที่สามารถแก้ไขได้ทุกขั้นตอน
2.5 Live Validation หรือ ระบบตรวจสอบตัวเลขแบบเรียลไทม์: สีเขียว (ถูกต้อง), สีแดง (ผิดกติกา) และ สีฟ้า (คำใบ้/Hint)
2.6 Data Persistence หรือ ระบบบันทึกเกม (Save/Load) คะแนนสถิติ ลงในไฟล์ .json
2.7 Responsive UI หรือ หน้าจอสลับระหว่างเมนูหลักและหน้าเล่นเกมได้ลื่นไหลด้วย ScreenManager

3. รายละเอียดเชิงเทคนิค 
3.1. Widgets มากกว่า 30 Widgets โดยโปรแกรมมีการใช้งาน Widget หลากหลายประเภท รวมแล้วเกิน 80 ชิ้น
3.1.1 TextInput (Sudoku Cells): จัดวางใน GridLayout (9x9) เพื่อรับข้อมูลจากผู้เล่น
3.1.2 Button: สำหรับควบคุมระบบต่างๆ เช่น Restart, Hint, Save, Load, Undo, Redo1
3.1.3 Multiple BoxLayout & GridLayout: ใช้ซ้อนกันเพื่อจัดระเบียบโครงสร้าง UI 
3.1.4 Label ทำ Stats Display : ใช้แสดงคะแนน เวลา และสถิติสูงสุดแบบสีสันสดใสด้วย MarkupPopup: สำหรับแจ้งเตือนชัยชนะ
3.2. Callbacks
มีการดักจับ Event สำคัญผ่านการใช้ bind และฟังก์ชันจัดการดังนี้
check_answer: ทำงานเมื่อมีการเปลี่ยนแปลงข้อความในช่องตาราง
start_new_game: เริ่มเกมใหม่และสุ่มโจทย์ตามระดับที่เลือก
toggle_note: สลับโหมดการป้อนข้อมูลระหว่าง Note และ Answer
undo_move: ย้อนกลับสถานะกระดานล่าสุด
redo_move: เดินหน้าไปยังสถานะที่เคยย้อนกลับไป
save_game: เขียนข้อมูลสถานะปัจจุบันลงไฟล์ JSON
load_game: อ่านไฟล์ JSON และกู้คืนกระดาน
give_hint: ระบบค้นหาตัวเลขที่ถูกต้องจาก Solution และเติมให้ผู้เล่น
clear_game: ล้างกระดานและรีเซ็ตเวลา
update_timer: ฟังก์ชัน Callback ที่ทำงานทุก 1 วินาทีผ่าน Clock.schedule_interval
go_to_menu: ระบบสลับหน้าจอกลับไปยังหน้าหลัก

4. อธิบายการทำงานของโค้ด
4.1 ตรรกะเบื้องหลัง (Backend Logic)
SudokuEngine: ทำหน้าที่สร้างกระดานที่ถูกต้อง โดยใช้หลักการ Recursive Backtracking และลบตัวเลขออกตามระดับความยากที่ผู้เล่นเลือก
Validation: ทุกครั้งที่มีการป้อนเลข ระบบจะเรียกฟังก์ชัน check_move เพื่อตรวจสอบว่าเลขนั้นซ้ำในแถว คอลัมน์ หรือบล็อก 3x3 หรือไม่
4.2 ส่วนติดต่อผู้ใช้ (Frontend Logic)
TextInput Binding: ทุกช่อง TextInput จะถูกผูกไว้กับฟังก์ชัน check_answer เมื่อผู้เล่นพิมพ์เลข โปรแกรมจะวิเคราะห์ทันทีว่าเป็น Note หรือ Answer แล้วจึงแสดงผลสีหรือขนาดฟอนต์ให้ถูกต้อง
Screen Management: ใช้ ScreenManager ในการจัดการลำดับชั้นของหน้าต่าง ทำให้แอปพลิเคชันดูเป็นระบบเหมือนแอปมือถือจริง
